## Getting Started

First, run the development server:

```bash
npm run dev

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Start by adding content to the page `app/page.js` in the src directory. The page auto-updates as you edit the file.

The layout file is located at `app/layout.js`. You can customize the layout by modifying this file. All pages will use this layout.

further instructions will follow.
```
